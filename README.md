## Happybob Project

Spelet är inspirerat av Flappy Bird och går ut på att styra en karaktär som ständigt faller nedåt. Spelaren trycker på space för att få karaktären att flyga uppåt och hålla sig i luften. Målet är att navigera genom hinder utan att krocka och att samla så hög poäng som möjligt genom att passera fler hinder.

## Kontroller
* Rörelse uppåt: Tryck på space för att få karaktären att hoppa eller flyga uppåt.
* Ingen annan rörelse: Karaktären rör sig automatiskt framåt i spelen.
## Funktionalitet
* Karaktären faller kontinuerligt nedåt på grund av gravitation, vilket gör att spelaren måste trycka på space med jämna mellanrum för att hålla karaktären i luften.
* Hinder (till exempel rör eller plattformar) är placerade längs spelbanan, och om karaktären krockar med något hinder avslutas spelet.
* Spelet räknar poäng baserat på hur många hinder spelaren lyckas passera utan att krascha.
* Svårighetsgraden ökar gradvis eftersom avståndet mellan hindren kan variera och hastigheten kan öka.
